## API Reference

#### Get all Links

```http
  GET /
```
#### Redirect to url

```http
  GET /${name}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `name`      | `string` | **Required**. Name of the Link |

## Run Locally

Clone the project

```bash
  git clone https://github.com/Red-Cafe/buyred-link-page
```

Go to the project directory

```bash
  cd buyred-link-page
```

Install dependencies

```bash
  npm install
```

Start the Project

```bash
  npm run start
```
